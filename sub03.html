Coremed

신뢰감을 주는 제약회사 브랜드 사이트

Duration: 2025.05 - 2025.06
Role: UI/UX Design, Web Publishing 100%
Tools: Figma, HTML, CSS, JavaScript
Visual
메인 화면에 가득 차있는 영상이 스크롤을 내리면 자동으로 90%의 비율로 작아지면서, 외곽은 라운드로 바뀌도록 하여 하단의 섹션과 잘 어우러지도록 jQuery로 구현하였습니다.
Section1
작은 이미지를 클릭하거나 호버하면 왼쪽 큰 섹션에 이미지가 보이고, 클릭하지 않더라도 자동으로 차례대로 보이도록 jQuery로 구현하였습니다. 스냅/영상 전문 업체인 브랜드의 특성에 맞춰 사진이 찍히는듯한 효과를 jQuery로 구현하였습니다.
Section2
사용자의 스크롤 컨트롤에 맞춰 이미지가 부드럽게 한 장씩 올라오는듯한 효과를 jQuery로 구현하였습니다.
Section3
Swiper-Slide를 사용하여 대표 사진을 넣고, 페이지네이션과 버튼을 이용하여 사용자가 사진을 쉽게 선택하여 볼 수 있도록 구현하였습니다. 또한, 사진을 직접 넘기지 않더라도 자동으로 무한 재생 가능하도록 infinity-loop 버전을 사용하였습니다.
Footer
연락처 정보와 소셜미디어 링크를 포함한 푸터로 사용자가 쉽게 연락할 수 있도록 구성했습니다.
Contact Form 코드 스크린샷
Header
포트폴리오 섹션은 업체의 특성에 맞춰 갤러리를 중심으로 구성했습니다. 메인과 동일한 브랜딩을 유지하면서도 갤러리에 최적화된 레이아웃을 적용했습니다.
Gallery Section
작품 이미지를 최적화하여 배치하고, 라이트박스 기능을 통해 사용자가 큰 화면으로 감상할 수 있도록 구현했습니다. 클라이언트가 새 이미지 파일들을 기존의 파일과 동일한 이름을 사용하여 업데이트하면 새로운 사진으로 업데이트 되도록 구현했고, 새로고침시 일정 비율로 랜덤 배치되도록 구성하여 감각적인 느낌을 더하였습니다.
Navigation
야외와 본식, 스냅과 필름 등 카테고리별로 작품을 분류하여 사용자가 원하는 스타일의 작품을 쉽게 찾을 수 있도록 구성했습니다.
Introduction Section
Brand Story는 업체 소개와 추구하는 방향성에 관한 정보를 담고 있습니다. About/Vision으로 섹션을 나누고, 각 페이지에서는 스토리텔링을 통해 브랜드의 가치와 철학을 담을 수 있도록 구성했습니다.
User Experience
하단 메뉴를 클릭했을때, 화면이 최상단으로 올라가지 않고 사용자가 보던 페이지에서 내용만 변경될 수 있도록 제작했습니다.
Responsive Design
모바일 환경에 최적화된 반응형 디자인을 적용했습니다. 터치 인터페이스와 스와이프 제스처를 고려한 UX를 구현했습니다.
Mobile Navigation
햄버거 메뉴를 활용한 모바일 네비게이션을 구현했습니다. 한 손 조작이 가능하도록 접근성을 고려한 UI를 설계했습니다.
Touch Optimization
터치 영역을 충분히 확보하고, 스와이프 갤러리와 터치 친화적인 인터랙션을 구현했습니다. 로딩 속도도 모바일에 최적화했습니다.
Performance
모바일 데이터 사용량을 고려한 이미지 최적화와 레이지 로딩을 적용했습니다. PWA 기능을 통해 앱과 같은 사용 경험을 제공합니다.
